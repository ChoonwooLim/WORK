🤖 텔레그램 봇 설정 방법

  1. @BotFather에게 /newbot 명령으로 봇 생성
  2. 받은 토큰을 .env 파일의 TELEGRAM_BOT_TOKEN에 입력
  3. @userinfobot에서 자신의 Telegram ID 확인
  4. ID를 TELEGRAM_ADMIN_ID에 입력
  5. 서버 재시작: pm2 restart remoteagt
