🚀 최신 업데이트

Orbitron에 새로 추가되거나 개선되는 모든 기능과 변경 사항을 한눈에 확인하세요.

v1.7.0 2026년 3월 6일

🌐 커스텀 도메인 자동 연결 & 도메인 관리 로드맵

  • 신기능
    커스텀 도메인 자동 연결 시스템

    가비아, Namecheap 등에서 구매한 도메인을 3단계(DNS 검증 → Cloudflare 터널 연결 → 활성화)로 자동 연결합니다. CNAME 레코드만 설정하면 SSL 인증서, 리버스 프록시, CDN이 모두 자동 적용됩니다.

  • 신기능
    DNS 자동 검증 API

    커스텀 도메인 연결 전 DNS 레코드가 올바르게 설정되었는지 실시간으로 자동 검증합니다. 잘못된 설정 시 구체적인 수정 가이드를 제공합니다.

  • 문서
    도메인 관리 가이드 & 구매 기능 로드맵

    가비아, Namecheap, Cloudflare 등 주요 도메인 등록기관별 CNAME 설정 가이드를 추가했습니다. 향후 도메인 검색/구매/자동 연동 기능(가비아 리셀러 API, Namecheap API)의 구현 로드맵을 공개했습니다.

v1.6.0 2026년 3월 3일

📱 카카오톡 및 텔레그램 모바일 시뮬레이터 & 통합 크로스 인증

  • 신기능
    SSE 탑재 텔레그램 및 카카오톡 모바일 웹 시뮬레이터

    실제 봇 토큰 없이도 브라우저 내에서 완벽한 프론트엔드 제어를 시연할 수 있도록 RemoteAGT 대시보드 내에 모바일 텔레그램 및 카카오톡 UI 구조를 완벽하게 모사한 웹 시뮬레이터를 런칭했습니다. 폴링 없는 실시간 SSE(Server-Sent Events) 비동기 파이프라인과 Webhook 모킹 기술을 통해 매끄러운 응답 경험을 선사합니다.

  • 신기능
    카카오톡 채널 자동 알림봇 연동

    이제 배포 진행 상황, 서버 장애 모니터링 경고 등을 텔레그램뿐만 아니라 국내 환경에 맞춘 '카카오톡 오픈빌더 스킬 연동'을 통하여 모바일 톡방에서 실시간으로 수신할 수 있습니다.

  • 개선
    Orbitron & RemoteAGT 크로스오버 통합 인증(SSO)

    이제 완전히 독립된 두 서비스가 물리적인 단일 PostgreSQL 데이터베이스 통신망 위에서 통합되었습니다. 어느 쪽에서 가입하든 *.twinverse.org 와일드카드 쿠키를 통해 완벽한 크로스 호환 로그인이 가능합니다.

  • 버그수정
    데이터베이스 Schema Column 누락 해결

    가입 시 활동 로그(ragt_audit_log) 로깅 과정에서 web_user_id 및 user_id 컬럼 부재로 발생하던 치명적 로그인 블록 버그를 완벽히 수정 및 마이그레이션 조치했습니다.

v1.5.0 2026년 3월 2일

🧠 Genius Deployer — 스마트 프로젝트 분석 & 멀티 서비스 배포

  • 신기능
    Smart Project Analyzer — 프로젝트 구조 자동 분석기

    배포 시 런타임(Node.js, Python, Go 등), 프레임워크(FastAPI, Vite, Next.js 등 20종+), 서비스 타입, 포트, 빌드 명령어, 의존관계를 100% 자동 감지합니다. orbitron.yaml 없이도 완벽한 배포가 가능합니다.

  • 신기능
    멀티 서비스 orbitron.yaml 지원

    하나의 레포에 백엔드, 프론트엔드, 직원앱 등 여러 서비스를 services 배열로 정의하면 각 서비스를 자동으로 올바른 타겟(Docker/CF Pages)에 배포합니다. from: 참조 문법으로 서비스 간 URL을 자동 연결합니다.

  • 신기능
    Cloudflare Pages 정적 사이트 자동 배포

    type: static 서비스를 감지하면 Cloudflare Pages CDN에 자동 빌드/배포합니다. 전 세계 에지 네트워크에서 초고속으로 서빙되며, 멀티 프론트엔드 동시 배포도 지원합니다.

  • 신기능
    PostDeploy 훅 — 배포 후 커스텀 명령 실행

    orbitron.yaml에 postDeploy: 배열을 정의하여 메인 배포 완료 후 추가 명령어를 자동 실행합니다. 환경변수 주입, 3분 타임아웃, 성공/실패 로그 기록을 지원합니다.

  • 버그수정
    정적 서비스 빌드 시 환경변수 미주입 수정

    Vite/React 정적 서비스 빌드 시 VITE_API_URL 등 환경변수가 주입되지 않아 프로덕션에서 localhost로 API를 요청하던 크리티컬 버그를 수정했습니다.

  • 문서
    배포 문서 5개 페이지 대규모 업데이트

    IaC 가이드, 정적 사이트, AI 분석기, 핵심 개념, 웹 서비스 문서에 멀티 서비스 배포, Smart Analyzer, CF Pages 자동 배포, Error Knowledge DB 내용을 추가했습니다.

  • 신기능
    Viewer 등급 — 읽기 전용 회원 등급 추가

    신규 가입자는 자동으로 'Viewer' 등급이 부여되어, 대시보드 메뉴를 둘러볼 수 있지만 프로젝트 생성·배포·수정은 불가합니다. 관리자가 등급을 승격(Viewer → User → Admin)할 수 있습니다.

v1.4.0 2026년 2월 28일

🤖 AI 에러 지식 데이터베이스 & 디스크 최적화

  • 신기능
    Error Knowledge DB — AI 에러 학습 시스템

    AI가 분석한 에러 패턴과 해결책을 자동 저장하여, 같은 에러 재발 시 즉시 과거 해결책을 제안합니다. 시간이 지날수록 더 빠르게 문제를 해결하는 학습형 배포 플랫폼입니다.

  • 개선
    Docker 이미지 HDD 마이그레이션

    Docker 이미지와 빌드 캐시를 SSD에서 HDD로 마이그레이션하여 SSD 디스크 공간을 최적화했습니다. 사이트 성능에 영향 없이 수십 GB의 공간을 확보합니다.

v1.3.0 2026년 2월 22일

📊 대시보드 리소스 모니터링 & 백업 강화

  • 신기능
    프로젝트 카드 실시간 리소스 통계

    대시보드 프로젝트 목록에서 CPU, 메모리, 업타임, 네트워크 I/O를 실시간으로 확인할 수 있습니다. 개별 페이지 진입 없이 한눈에 모든 프로젝트를 모니터링합니다.

  • 신기능
    외부 DB 백업 지원 (DATABASE_URL 기반)

    PostgreSQL 백업 기능이 확장되어, DATABASE_URL 환경변수로 연결된 외부 데이터베이스도 원클릭 백업이 가능합니다.

  • 신기능
    미디어 파일 백업 서비스

    프로젝트 내 미디어 파일(이미지, 문서 등)을 DATA 드라이브에 자동 백업합니다. 수동 백업/복원 기능과 배포 시스템 연동을 제공합니다.

  • 신기능
    Git 레포 로컬 클론 백업

    GitHub URL 옆 'Clone' 버튼으로 레포를 로컬 백업 디렉토리에 클론합니다. 이미 존재하면 git pull로 업데이트됩니다.

v1.2.0 2026년 2월 21일

🎯 커밋 선택 배포 & AI 채팅 개선

    요금제 페이지 신규

    플랜별 가격, 컴퓨트 비용, 기능 비교 표, 인터랙티브 FAQ 아코디언이 포함된 요금제 페이지를 추가했습니다.

v1.1.0 2026년 2월 20일

🏗 초기 프로덕션 안정화

  • 신기능
    GitHub Webhook 자동 배포 (CI/CD)

    GitHub 레포지토리에 Push하면 자동으로 감지하여 무중단 배포를 실행합니다. Blue-Green 스위칭으로 다운타임 제로를 보장합니다.

  • 신기능
    AI 에러 분석 어시스턴트

    배포 실패 시 Claude/Gemini AI가 에러 로그를 분석하여 한국어로 원인과 해결책을 제시합니다.

  • 신기능
    Unreal Engine 픽셀 스트리밍

    듀얼 GPU 로드밸런싱이 적용된 Unreal Engine 픽셀 스트리밍 서버를 원클릭 배포합니다.

  • 보안
    Cloudflare Tunnel 자동 보안 인그레스

    모든 배포 프로젝트에 Cloudflare Tunnel을 통한 HTTPS 보안 접속과 DDoS 방어를 자동 적용합니다.
