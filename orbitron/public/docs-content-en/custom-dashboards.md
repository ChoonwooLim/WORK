# Custom Dashboards

> 💡 **Coming Soon**
> 
> This feature is a core roadmap item currently under active development by the Orbitron engineering team.
> 

In large projects operating dozens of microservices (MSA), manually clicking into each individual card to view information is a laborious task. 

For development leads or CTOs, this feature allows you to **select only the desired metric widgets, like LEGO blocks, and display them prominently on the main screen like a dashboard.**

*   **Payment API Widget:** Server Response Time (Latency) Monitor Dashboard
*   **Frontend Widget:** Real-time Chart for 404 Error Occurrence Frequency
*   **DB Widget:** PostgreSQL Connection Pool Potential Saturation Monitor
