# Custom Dashboards

> 💡 **Coming Soon (Upcoming Feature)**
> 
> This is a core roadmap feature currently in intense development by the Orbitron engineering team.
> 

In massive projects running dozens of microservices (MSA), clicking into individual card information one by one is manual labor.

This is a feature designed for team development leads or CTOs, allowing them to **pick out only desired Metric widgets like Lego blocks and stick them on the main screen like an electronic billboard**.

*   **Payment API Widget:** Server response time (Latency) monitor billboard
*   **Frontend Widget:** Real-time chart of 404 error occurrence frequency
*   **DB Widget:** Monitor for PostgreSQL Connection Pool saturation capacity
