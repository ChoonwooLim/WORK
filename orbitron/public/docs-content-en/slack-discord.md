# 배포 상태 알림 (Slack, Discord 웹훅)

> 💡 **출시 예정 기능 (Coming Soon)**
> 
> 이 기능은 현재 Orbitron 엔지니어링 팁에서 열심히 개발 중인 핵심 로드맵 기능입니다.
> 

대시보드 화면을 하루 종일 쳐다보고 있을 순 없습니다. 코드가 Push되고 빌드가 돌아갔을 때의 결과를 팀 전체가 실시간 메신저로 받아보게 해주는 콜백(Callback) 연동 기능입니다.

*   **배포 성공/실패 즉시 알림:** 여러분의 팀이 쓰고 있는 Slack 워크스페이스나 Discord 채널로 "🎉 결제 서버 v1.02 배포 완료" 또는 "🚨 [실패] 프론트엔드 모듈 설치 중 충돌" 메시지가 아주 깔끔한 카드 형태로 꽂힙니다.
*   **커스텀 Webhook 발송:** 자체적으로 만드신 사내 시스템 연동을 위하여, 빌드 라이프사이클 훅에 맞춰 아무 HTTP 엔드포인트로나 상태 JSON 데이터를 `POST` 전송할 수 있도록 지원합니다.
