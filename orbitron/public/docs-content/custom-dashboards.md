# 커스텀 관제 대시보드 (Custom Dashboards)

> 💡 **출시 예정 기능 (Coming Soon)**
> 
> 이 기능은 현재 Orbitron 엔지니어링 팁에서 열심히 개발 중인 핵심 로드맵 기능입니다.
> 

수십 개의 마이크로서비스(MSA)를 굴리는 대형 프로젝트에서는 개별 카드 정보를 일일이 클릭해 들어가는 것이 노동입니다. 

팀의 개발 책임자나 CTO를 위해, **원하는 지표(Metric) 위젯들만 레고 블록처럼 골라내어 메인 화면에 전광판처럼 붙여놓을 수 있는** 기능입니다. 

*   **결제 API 위젯:** 서버 응답시간(Latency) 모니터 전광판
*   **프론트엔드 위젯:** 404 에러 발생 빈도 실시간 차트 
*   **DB 위젯:** PostgreSQL 커넥션 풀(Pool)포화 가능성 모니터
