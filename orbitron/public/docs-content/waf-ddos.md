# WAF 및 L7 DDoS 실시간 방어망

> 💡 **출시 예정 기능 (Coming Soon)**
> 
> 이 기능은 현재 Orbitron 엔지니어링 팁에서 열심히 개발 중인 핵심 로드맵 기능입니다.
> 

현재 Orbitron이 기본 제공하는 L3/L4 네트워크 계층의 방어벽(Cloudflare Tunnel)과 더불어, 애플리케이션 계층(L7)을 노리는 더욱 지능적인 웹 공격을 원천 차단하는 고급 엔터프라이즈 기능입니다.

*   **웹 애플리케이션 방화벽 (WAF):** 악의적인 SQL Injection(데이터베이스 뚫기 시도)이나 Cross-Site Scripting(XSS) 같은 교묘한 악성 페이로드가 포함된 HTTP 요청을 최전선에서 탐지하여 자동으로 차단합니다.
*   **스마트 봇(Bot) 차단:** 사람인 척 위장하여 사이트의 귀중한 데이터를 미친 듯이 긁어가는 악성 크롤링 봇이나 트래픽 유발 매크로를 분석하여 접속 대역폭을 막아버립니다. 
*   **IP 차단/허용 리스트 (Access Control):** 중국이나 러시아 등 아예 특정 국가의 접속 트래픽을 원천적으로 막거나, 사내 IP 주소 대역에서만 개발 중인 서비스에 들어올 수 있도록 White-list를 대시보드에서 설정할 수 있습니다.
