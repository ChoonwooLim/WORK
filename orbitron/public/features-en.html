Features for the Ultimate Developer Experience

🎮

Unreal Engine Streaming

Upload your Linux dedicated server ZIP, and we'll instantly configure a Pixel Streaming web viewer and matchmaking server with auto dual-GPU load balancing.

🎲

Blazing Fast Unity WebGL

Enjoy a dedicated deployment ecosystem maximizing WebAssembly (WASM) parsing and loading speeds through pre-compressed Gzip/Brotli support and Nginx optimizations.

🤖

AI-Powered Diagnostics

Upon deployment failure, our triple LLM routing (Claude / Gemini / Local Gemma 4) analyzes logs and context, instantly providing markdown guides for the cause and solutions. Per-project model selection lets you choose between maximum cloud reasoning power and zero-cost, fully-offline local inference for privacy-sensitive workloads.

🌱

Local Gemma 4 (Ollama, New)

Run Google DeepMind's Gemma 4 E4B directly on your local GPU (CUDA) for AI debugging with zero external API cost, no internet connection required, and your data never leaves the server. No Anthropic/Google signup needed — works out of the box. Model data is isolated to a separate data drive to keep system disk usage low, and the Ollama daemon is persisted via systemd user units for automatic startup on login. (2026.04 v2.1)

🎬

AI Video Generation (Wan 2.2, New)

Three Alibaba Wan 2.2 official open-source variants (TI2V-5B / T2V-A14B / I2V-A14B) self-hosted on a dedicated RTX 3090 GPU server. Generate prompt → MP4 or still → animation right from the dashboard — zero external API cost, no per-token billing. The 5B model renders in ~36s; the A14B MoE flagship fits on a single 24GB GPU via sequential CPU offload. Videos stream back through Orbitron's authenticated proxy (/api/wan/video/*), so the LAN IP never leaks to the browser. (2026.04 v2.2)

🖥

Distributed GPU Inference (New)

The production dashboard host is decoupled from a dedicated GPU server (Threadripper 3970X + RTX 3090 24GB + 128GB RAM). Heavy AI workloads (31B LLMs, A14B video MoE) live on the GPU box; restarting the dashboard doesn't interrupt a running render. Adding more GPU boxes is just an env-var change (OLLAMA_HOST / WAN_VIDEO_HOST). Three concurrent workloads on one GPU are kept from OOM by a triple guard (Ollama 30s idle unload, Wan-service pre-load hook, diffusers sequential offload). (2026.04 v2.2)

🔄

Infinite GitHub CI/CD

Simply enter your repository URL and push. Orbitron instantly detects changes, rebuilds your Docker containers, and rolls them out to live services without complex configuration.

🌐

Cloudflare Ingress Security

No need for complex router port forwarding. Upon creation, a unique Cloudflare Edge Tunnel is generated, securely tunneling all HTTP/Dev ports to the internet.

📁

Hybrid Backup Storage

Fully supports hardware mounting paths to perfectly prevent data loss. Archive and compress all volumes' media and source code with a single click.

⚡

Zero-Downtime Deployments

Never experience a single second of downtime during app updates. The new version runs flawlessly in the background before Nginx proxy routing instantly snap-switches (Blue-Green) without user interruption.

🐳

Multi-Container Orchestration

Don't just launch a single web app. If a docker-compose.yml file is detected, dozens of complex containers—Node.js, Redis, Celery Workers—are orchestrated and deployed together as one cohesive system.

🧠

Smart Project Analyzer

Automatically detects runtimes (Node.js, Python, Go, etc.), frameworks (FastAPI, Vite, Next.js, etc. 20+), service types, ports, build commands, and inter-service dependencies. Deploy perfectly even without orbitron.yaml.

☁️

Cloudflare Pages Auto-Deploy

When a type: static service is detected, it's automatically built and deployed to Cloudflare Pages CDN. Supports build-time env injection and multi-frontend simultaneous deployments across global edge network.

📦

Multi-Service Orchestration

Define backend, frontend, and staff apps in a single services array. Use from: references to auto-link service URLs, with dependency-based deployment ordering.

🪝

PostDeploy Hooks

Automatically execute custom commands after main deployment completes. Define CF Pages deploys, DB seeding, Slack notifications in postDeploy: with env injection.

📚

Error Knowledge DB

A self-learning database that stores AI-analyzed error patterns and solutions. On recurring errors, past solutions are instantly suggested—getting smarter over time.
