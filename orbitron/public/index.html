최고의 개발 경험을 위한 기능들

🎮

Unreal Engine 스트리밍

리눅스 전용 빌드 ZIP 파일만 업로드하면, 자동 듀얼 GPU 로드밸런싱과 함께 Pixel Streaming 웹 뷰어와 매치메이킹 서버를 즉시 구성합니다.

🎲

Unity WebGL 광속 호스팅

Gzip/Brotli 사전 압축 지원 및 Nginx 최적화 적용으로, 웹 어셈블리(WASM) 패키지의 파싱 및 로딩 속도를 극대화한 전용 배포 생태계를 제공합니다.

🤖

AI 기반 에러 분석

배포 실패 시, 시스템 통합된 Claude 및 Gemini 모델이 에러 로그를 읽고 컨텍스트를 파악해 1초 만에 원인과 해결책 가이드를 마크다운으로 제시합니다. (현재 코드를 직접 강제 수정하지는 않으며, 안전한 가이드만 제공합니다.)

🔄

GitHub 무한 CI/CD

별도의 설정 없이 저장소 URL을 입력하고 Push 해보세요. 코드가 올라옴과 동시에 Orbitron이 감지하여 Docker 컨테이너를 재빌드하고 라이브 서비스로 롤아웃합니다.

🌐

Cloudflare 인그레스 보안

복잡한 공유기 포트포워딩은 필요 없습니다. 시스템 생성과 동시에 고유한 Cloudflare Edge 터널을 생성하여 모든 HTTP/Dev 포트를 인터넷에 안전하게 터널링합니다.

📁

하이브리드 백업 스토리지

데이터 유실을 완벽히 방어하기 위해 DATA 드라이브 하드웨어 마운트를 완벽히 지원하며, 클릭 한 번으로 모든 볼륨의 미디어 및 소스 코드를 아카이브 압축합니다.

🗄

매니지드 데이터베이스

PostgreSQL과 Redis 엔진을 원클릭으로 가동합니다. 별도 설정 없이 랜덤 비밀번호와 안전한 Connection String을 즉시 발급받아 웹앱과 연동하세요.

🔒

프라이빗 네트워크

배포되는 모든 인프라 자원을 `orbitron_internal` 전용 브릿지망에 자동 묶어, 외부 클라우드플레어 터널링 패스 없이 데이터베이스와 초고속 보안 통신을 보장합니다.

⚙️

백그라운드 워커

디스코드 봇, 크론(Cron) 스케줄러, 비동기 파이프라인 등 퍼블릭 포트 오픈이 필요 없는 무한 백그라운드 프로세스를 조용히 그리고 끈질기게 구동시킵니다.

📜

Infrastructure-as-Code

대시보드 조작 없이 레포지토리 내 정의된 단일 `orbitron.yaml` 코드를 시스템이 파싱하여 환경 변수, 포트 번호, 빌드 커맨드 설정을 즉각 덮어씌웁니다.

🚀 일반 앱 배포하기 (GitHub / ZIP 업로드)

▼

1. GitHub 연결을 통한 배포

1. 대시보드 로그인 후 + New Project 버튼을 클릭합니다.
2. "GitHub" 탭에서 레포지토리 URL을 입력하고 "자동 배포" 토글을 켭니다.
3. npm install && npm run build 등의 빌드 커맨드를 지정하고 생성합니다.
4. 배포가 완료되면 대시보드 우측의 Webhook URL을 복사하여 GitHub 저장소 설정(Webhooks)에 등록하면, 향후 Push 발생 시 자동 재배포됩니다.

2. 단일 폴더 (ZIP) 업로드 방식

1. 프로젝트 최상위 파일을 모두 선택하고 압축한 ZIP 파일을 준비합니다. (가장 바깥 폴더를 압축하지 말고 내용물 전체를 드래그해 압축)
2. "일반 앱 업로드" 모드에서 ZIP 파일을 드롭존에 넣습니다.
3. 포트를 비워두면 시스템이 가용 포트를 자동 할당하여 컨테이너를 구동합니다.

🎮 Unreal Engine 픽셀 스트리밍 서버 구축

▼

Orbitron은 리눅스 기반 Unreal Matchmaker 배포 템플릿을 자동 생성합니다.

1. 패키징 요구사항

Unreal Engine 편집기에서 빌드 시, 타겟 플랫폼을 반드시 Linux > Server로 설정하여 패키징 하십시오.

2. 대시보드 업로드

1. 좌측 사이드바 전용 배포 란에 있는 Unreal Engine 배포를 클릭합니다.
2. 게임의 영문 이름을 입력합니다. (예: mygame)
3. 방금 추출한 리눅스 서버 루트 폴더를 ZIP 파일로 압축하여 올립니다.
4. 백그라운드에서 Vulkan 도구가 셋업되고 GPU 포트 맵핑이 끝나면 "🎮 게임 시작" 버튼이 뜹니다. 해당 버튼 누를 시 외부 접속 지원 웹 매치메이커 클라이언트로 이동하게 됩니다.

🎲 Unity WebGL 퍼블리싱 가이드

▼

1. WebGL 빌드 폴더 준비

유니티에서 WebGL로 빌드하면 나오는 index.html, Build, TemplateData 폴더 전체를 선택하여 하나의 ZIP으로 압축합니다.

2. 원클릭 호스팅

1. 좌측 사이드바 Unity WebGL 배포를 클릭합니다.
2. 압축 파일을 올립니다. Orbitron의 내부 프로비저닝 로직은 이 프로젝트만을 위한 정적 특화 Nginx Alpine 컨테이너를 가동하여 사전 압축된 WebAssembly 메타를 처리합니다.
3. 브라우저에서 Unity MIME 경고 없이 초고속으로 렌더링 됩니다.

💬 AI 모델 키 연동 및 적용법

▼

Orbitron의 핵심 컴포넌트인 "에러 자동 분석 및 챗봇" 기능을 사용하시려면, AI API Key를 프로젝트별로 등록하셔야 합니다.

1. 대시보드 -> 프로젝트 클릭 후 설정(Settings) 탭으로 이동
2. 🤖 AI 에러 분석 모델 설정의 드롭다운 선택상자에서 사용할 모델 (Claude 4.6 시리즈 또는 Gemini 2.5/3.0 시리즈 등)을 선택합니다.
3. 발급받기 버튼을 눌러 Anthropic 콘솔 또는 Google AI Studio에서 무료 API Key를 확보합니다.
4. Key를 저장하면 이제 배포가 실패할 때마다 시스템이 로그를 읽고 원인을 추론하여 띄워주며, 사이드바의 AI 어시스턴트 탭에서 대화가 가능해집니다.
